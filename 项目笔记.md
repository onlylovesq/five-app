## 步骤

#### 1.头部组件布局及 shadcn 安装

Header布局使用 tailwind.css，其中涉及到了一个 @layer componnets 语法，这个语法写在了 globals.css中

```css
@layer components {
  .container {
    @apply w-[1140px] mx-auto;
  }
}
```

解释下这个语法前，先说 tailwind 的分层

Tailwind 将 CSS 分为三个主要的“层（Layer）”，顺序非常重要：
+ Base (基础层): 重置样式（Reset CSS），HTML 标签的默认样式。
+ Components (组件层): 你的这段代码就放在这里。用于定义按钮、卡片、容器等复杂的 UI 组件。
+ Utilities (工具层): 像 p-4, text-center, flex 这种单一功能的原子类。

在 CSS 中，后定义的样式会覆盖先定义的样式。Tailwind 强制规定 Utilities（工具类） 必须在 Components（组件类） 后面加载

加了 @layer components：Tailwind 会确保生成的 CSS 中，.container 排在 w-full 前面。这样，你在 HTML 里加的任何工具类（如 w-full, mt-4）都能成功覆盖 .container 里的默认样式。

还有就是为了实现头部右边导航的分割线，我们使用 shadcn 这个库，这个库的好处是，每次使用一个组件都要安装，这就从根本上实现了按需加载，安装好后它会把组件代码生成到你的 components/ui 这个目录下，你可以引用也可以手动修改

```shell
# 安装 shadcn
npx shadcn@latest init -d
# 安装具体组件
npx shadcn@latest add separator
```
